1. Создайте Telegram-бота через BotFather и получите токен.
2. В Heroku:
    - Зарегистрируйтесь через GitHub (если ещё нет аккаунта).
    - Создайте новый проект → New Project → Deploy from GitHub (если зальёте туда код) или просто загрузите файлы.
    - Установите переменную окружения BOT_TOKEN со своим токеном бота.
    - Запустите проект.
3. Бот будет онлайн 24/7 бесплатно (пока не превысите лимиты Heroku Free Plan).
4. Если нужно будет что-то изменить (например, добавить новые склады) — просто обновляйте warehouses.json и перезапускайте бота.
